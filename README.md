# Seymour Mining Suite App Store

A curated mining infrastructure platform built on Umbrel.

## Featured Seymour Applications

- Nexus Command Center
- Seymour MiningCore
- Seymour Backup Manager

## Blockchain

- Bitcoin Core
- Bitcoin Cash Node

More mining-focused chains will be added over time.

## Infrastructure

- PostgreSQL
- Redis
- Tailscale
- Cloudflared
- Nginx Proxy Manager

## Philosophy

If Nexus Command Center can do it well, we do not add another app just to add another app.

Nexus is the control room.
Umbrel is the platform.
Seymour Mining Suite is the mining ecosystem.
